# Requirements #

VIM >= 7.2.

# Installation #

Please note that running ./symlink.sh will **delete** any existing ~/.vim/, ~/.vimrc and ~/.gvimrc. 
**You have been warned.**

    git clone git://github.com/omh/dotfiles.git ~/.dotfiles
    cd ~/.dotfiles
    ./install.sh
    ./symlink.sh
